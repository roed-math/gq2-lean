## 1. Route — 𝔽₂-native, replacing the paper's 𝔽̄₂/Clifford machinery

The paper proof (p. 29): pass to `𝔽̄₂`, Clifford theory on an absolutely irreducible
constituent, "every χ-weight space is one-dimensional", 2-power-order scalars are trivial,
so a Sylow-2 `P` acts freely on weights in blocks of `2^a`; regular blocks; descend.
**Do not formalize that.**  The same content falls out 𝔽₂-natively from the repo's own
P-15f3 pattern (`UnramifiedModel.cCoeff_faithful_and_simple`): build the *inertia subring*
and exploit that **conjugation by Frobenius is literally the ring Frobenius** on it — the
tame relation `s⁻¹ t s = t²` says so.  The weight-stabilizer step then becomes a short
eigenvector-orbit computation, no algebraic closure, no induced modules, no weight spaces.

Notation: `s := c tameSigma`, `t := c tameTau`, `act g := DistribSMul.toAddMonoidHom`-style
`g`-action in `AddMonoid.End V`, `θ := act t`.

### Brick B — the inertia field `F` (~300 ln; mirror of f3's Schur section)

`A := Subring.closure {θ}` in `AddMonoid.End V`.

* `A` is commutative (singleton closure — reuse f3's technique, `UnramifiedModel.lean:184–273`)
  and char 2 (`hV2` ⟹ `1 + 1 = 0` in `End V`).
* **Conj-is-Frobenius**: `conj_s : e ↦ act s⁻¹ ∘ e ∘ act s` is a ring endo of `End V`;
  on the generator `conj_s θ = act (s⁻¹*t*s) = act (t^2) = θ²` (use `act (g*h) = act g ∘ act h`).
  Two ring homs `A → End V` (`conj_s` and Frobenius `a ↦ a²` — a ring hom since `A` comm char 2)
  agree on `{θ}` ⟹ agree on the closure (equalizer-subring or `Subring.closure_induction`).
  So `∀ a ∈ A, conj_s a = a ^ 2`; `Frob : A → A` is **bijective** (conjugation is injective;
  `A` finite and `a² ∈ A`).
* **Nonzero elements are bijective** (the new *stabilized-kernel trick* — this replaces
  f3's orbit-pinning): for `0 ≠ a ∈ A`, the chain `ker (a^(2^j))` stabilizes at some `J`;
  `K∞ := ker (a^(2^J))` is
  `t`-stable (`A` comm), and `s`-stable: `a^(2^J) ∘ act s = act s ∘ conj_s (a^(2^J)) =
  act s ∘ a^(2^(J+1))`, and `K∞ = ker a^(2^(J+1))` (stabilized!) — this is why the trick
  works.  Generator-stability + finite order ⟹ `C`-stability ⟹ `hsimple K∞`.
  `K∞ = ⊤` would give `a^(2^J) = Frob^J a = 0` ⟹ `a = 0` (Frob injective) ✗.
  So `K∞ = ⊥` ⟹ `ker a = ⊥` ⟹ `a` bijective on finite `V`.
* **`A` is a field `F`**: inverses inside `A` by power-cycling (`a^i = a^j`, `a` injective ⟹
  `a^(j−i) = 1` ⟹ `a⁻¹ = a^(j−i−1) ∈ A`); nontriviality from `V` nontrivial (`hram` witness
  forces `Nontrivial V`: `t • v ≠ v` fails on subsingletons).  Package as `IsField` /
  `Field ↥A` (`IsField.toField`); f3's `Finite.isDomain_to_isField` also works.
* `Module ↥F V` by `⟨a,_⟩ • v := a v` (laws are `End`-tautologies); `Module.Finite` from
  `Finite V`.  `θ ∈ F` is a unit of odd multiplicative order (divides `orderOf t`, odd by A).

### Brick D — the crux: twist order = `2^a` (~250 ln)

This is the paper's "stabilizer acts by a 2-power scalar, hence trivially", de-Clifford-ized.

* **Twist classification**: `S := {g : C | ∃ j, ∀ a ∈ A, conj_g a = Frob^[j] a}` is a
  subgroup (mul: compose/add; inv: `Frob` has finite order on finite `A`) containing `s`
  (`j = 1`, Brick B) and `t` (`j = 0`, `θ ∈ A` comm).  `hgen` ⟹ `S = ⊤`.  Extract
  `j : C → ℕ` and `hj g : ∀ a ∈ A, conj_g a = Frob^[j g] a`.
* `φ := Frob^[j p]` restricted to a ring auto of `F`; `m := orderOf φ` (as `F ≃+* F`,
  or as a permutation — finite).  Set `q := p ^ m`: then `conj_q = id` on `A`, i.e.
  **`act q` commutes with every `a ∈ F`** (F-linear).
* **`q = 1`** (the whole game):
  1. `q ∈ ⟨p⟩` has 2-power order `2^c`; char-2 helper `(x − 1)^(2^n) = x^(2^n) − 1` for
     commuting `x` (tiny induction) gives `(act q − 1)^(2^c) = act (q^(2^c)) − 1 = 0`.
  2. Nilpotent on nontrivial finite `V` ⟹ `∃ v ≠ 0, q • v = v` (minimal-`n` witness).
  3. For **every** `g : C`: `C ⧸ zpowers t` abelian ⟹ `g⁻¹ * q * g = t ^ m' * q` for
     some `m' : ℕ` (reduce mod `orderOf t` to keep exponents in ℕ).  Then
     `q • (g • v) = g • ((t^m' * q) • v) = g • (θ^m' v) = λ • (g • v)` where
     `λ := Frob^[j g⁻¹] (θ^m') ∈ ⟨θ⟩ ⊆ F` — the semilinear pass-through
     `act g ∘ a = (conj_{g⁻¹} a) ∘ act g`.  So each orbit vector `g • v` is a
     `λ`-eigenvector of `act q` with `λ` of **odd** multiplicative order.
  4. `act q` commutes with `λ` ⟹ `(λ^(2^c) − 1) (g • v) = 0` with `g • v ≠ 0`; `F` a field
     acting on `V` ⟹ `λ^(2^c) = 1`; odd order ∧ 2-power exponent ⟹ `λ = 1`.
     So `q` fixes the whole `C`-orbit of `v`.
  5. `W := AddSubgroup.closure (Set.range (· • v))` is `C`-stable and nonzero ⟹
     `hsimple` ⟹ `W = ⊤`; `act q` additive and fixes the generators ⟹ `act q = 1` ⟹
     `q = 1` by `hfaith`.
* Hence `orderOf p ∣ m`; conversely `φ^(2^a) = conj_{p^(2^a)} = conj_1 = id` ⟹ `m ∣ 2^a`.
  **`m = 2^a` exactly.**

### Brick D′ — the semilinear count `|V^p|^(2^a) ≤ |V|` (~150 ln)

* `g₀ := act p` is `φ`-semilinear: `p • (α • v) = φ α • (p • v)` (rearranged `hj`).
* `F₀ := FixedPoints.subfield (Subgroup.zpowers φ-in-(F ≃+* F)) F`; membership ⟺ `φ x = x`.
  `Module.finrank ↥F₀ F = 2^a` by `FixedPoints.finrank_eq_card` ✓(verified:
  `[MulSemiringAction G F] [Fintype G] [FaithfulSMul G F]`; faithfulness of `zpowers φ` on
  `F` from `orderOf φ = m`).
* `V₀ := {v // p • v = v}` is an `F₀`-module (semilinearity fixes `F₀`-scalars).
* **Artin independence**: an `F₀`-independent family in `V₀` is `F`-independent in `V` —
  minimal-support relation, apply the semilinear `g₀`, subtract; the support shrinks unless
  all coefficients are `φ`-fixed.  (Self-contained; state over `LinearIndependent`.
  Fallback formulation if the two-module-structure plumbing fights: induct on a
  `Finset`-indexed family and conclude the `finrank` inequality directly.)
* Conclude `finrank F₀ V₀ ≤ finrank F V =: k`, then cards via

  `|V₀| = |F₀|^{r₀} ≤ |F₀|^k`, `|V| = |F|^k = |F₀|^{2^a·k}` ⟹ **`|V^p|^(2^a) ≤ |V|`**.

### Brick E — freeness and the explicit `P`-package (~250 ln)

Work over `R := ZMod 2` (`AddCommGroup.zmodModule`), `u := act p − 1` as an `R`-linear endo
(`AddMonoidHom.toZModLinearMap`-or-hand-rolled — *verify name at execution*).

* `u^(2^a) = 0` (char-2 helper + `p^(2^a) = 1`).
* **Jump dimensions**: `δ j := finrank R (ker u^(j+1)) − finrank R (ker u^j)` is antitone
  in `j` (`u` induces an injection `ker u^(j+1) ⧸ ker u^j →ₗ ker u^j ⧸ ker u^(j−1)`), and
  `Σ_{j<2^a} δ j = finrank R V`, `δ 0 = finrank R V^p`.  With D′
  (`2^a · finrank V^p ≤ finrank V`): `finrank V ≤ 2^a · δ 0 ≤ finrank V` ⟹ all `δ j = δ 0 =: κ`.
  In particular `im (u^(2^a −1)) ⊆ V^p` with equal finrank ⟹ **`im u^(2^a−1) = V^p`**
  (the norm criterion, proved by counting — no PID structure theorem, no
  `Module.equiv_directSum_of_isTorsion`).
* Pick an `R`-basis `b : Fin κ → V^p` and preimages `v i` with `u^(2^a−1) (v i) = b i`.
* `Θ : (Fin κ → ↥P → ZMod 2) →+ V`, `Θ F := Σ_i Σ_{x : ↥P} F i x • (x • v i)` — the same
  formula as `RegularSummand.equivariant_lift_of_regular_summand`'s `G`; copy-adapt its
  equivariance calc (~15 ln; do NOT import/edit the untracked file, just reuse the shape).
* `Θ` **surjective**: `im Θ = ` the `𝔽₂`-span of `{x • v i}` ⊇ span `{u^j (v i)}`
  (binomial: `u^j` is an ℕ-combination of `act p^l`), and that span is `⊤` by *downward
  induction on kernel level*: `w ∈ ker u^(m+1)` ⟹ `u^m w ∈ V^p = span {u^(2^a−1)(v i)}`
  ⟹ subtract `Σ cᵢ u^(2^a−1−m)(v i)` and recurse.  Cards match
  (`2^(κ·2^a) = |V|` from the jump equality) ⟹ **bijective**
  (`Nat.bijective_iff_injective_and_card`-family).
* Package: `r_P := Θ`, `ι_P := (AddEquiv.ofBijective Θ …).symm`; equivariance of the
  inverse is mechanical from `Θ`'s.
